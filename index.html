טיפים מהירים

  • לחץ על Ctrl + C (או Cmd + C במק) כדי להעתיק טקסט מסומן
  • השתמש ב-Alt + Tab כדי לעבור בין חלונות פתוחים
  • לחץ על Win + D (או Cmd + F3 במק) כדי להציג את שולחן העבודה
  • השתמש ב-Ctrl + F (או Cmd + F במק) כדי לחפש בדף או במסמך
  • לחץ על Win + L (או Ctrl + Cmd + Q במק) כדי לנעול את המחשב במהירות
