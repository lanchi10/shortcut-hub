חזרה לדף הראשי

קיצורי דרך לכלי פרודוקטיביות

Microsoft Office

  • Ctrl + C: העתק
  • Ctrl + X: גזור
  • Ctrl + V: הדבק
  • Ctrl + Z: בטל פעולה אחרונה
  • Ctrl + Y: בצע שוב פעולה שבוטלה
  • Ctrl + B: הדגש טקסט
  • Ctrl + I: הטה טקסט
  • Ctrl + U: קו תחתון
  • Ctrl + F: חפש במסמך

Google Sheets

  • Ctrl + Space: בחר עמודה
  • Shift + Space: בחר שורה
  • Ctrl + Shift + V: הדבק ערכים בלבד
  • Ctrl + ;: הכנס תאריך נוכחי
  • Ctrl + Shift + :: הכנס שעה נוכחית
  • Alt + Shift + F: הוסף פילטר
  • Ctrl + Alt + E: פתח רשימת כלים

Slack

  • Ctrl + K: פתח חיפוש מהיר
  • Ctrl + /: פתח רשימת קיצורי דרך
  • Ctrl + Shift + K: פתח שיחה ישירה
  • Ctrl + Shift + M: סמן הכל כנקרא
  • Ctrl + [: נווט אחורה בהיסטוריה
  • Ctrl + ]: נווט קדימה בהיסטוריה
  • Ctrl + Shift + Y: התחל שיחת וידאו

Trello

  • B: פתח את רשימת הלוחות שלך
  • C: ארכב כרטיס
  • D: פתח חלון ארכיון ללוח הנוכחי
  • F: פתח חלון חיפוש
  • M: הקצה חבר צוות לכרטיס
  • N: צור כרטיס חדש תחת רשימה
  • Q: הצג רק כרטיסים שהוקצו לך

Notion

  • Ctrl + Shift + N: פתח חלון חדש של Notion
  • Ctrl + P: חפש או עבור לדף
  • Ctrl + /: פתח תפריט הפקודות
  • Ctrl + Shift + Arrow: הזז בלוק למעלה/למטה
  • /todo: הוסף משימה
  • /h1: צור כותרת ראשית
  • /date: הוסף תאריך נוכחי

טיפים לשימוש יעיל בקיצורי דרך בכלי פרודוקטיביות

  • השתמש בקיצורי דרך באופן קבוע כדי להפוך אותם להרגל.
  • התאם אישית קיצורי דרך בתוכנות שמאפשרות זאת לפי הצרכים שלך.
  • שקול להשתמש בכלי מקרו ליצירת קיצורי דרך מורכבים יותר.
  • למד את הקיצורים הספציפיים לכל תוכנה שאתה משתמש בה באופן קבוע.
