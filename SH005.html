חזרה לדף הראשי

קיצורי דרך לעריכת מדיה ופרזנטציות

Adobe Photoshop

  • Ctrl + T: טרנספורמציה חופשית
  • Ctrl + J: שכפל שכבה
  • Ctrl + [: הורד שכבה אחת למטה
  • Ctrl + ]: העלה שכבה אחת למעלה
  • Ctrl + Alt + I: שנה גודל תמונה
  • Ctrl + Shift + N: שכבה חדשה

Canva

  • Ctrl + C: העתק אלמנט
  • Ctrl + V: הדבק אלמנט
  • Ctrl + Z: בטל פעולה אחרונה
  • Ctrl + G: קבץ אלמנטים
  • Ctrl + Shift + G: פרק קבוצה
  • Ctrl + K: הוסף קישור

PowerPoint

  • Ctrl + M: הוסף שקופית חדשה
  • Ctrl + D: שכפל שקופית
  • F5: הפעל מצגת ממתחילתה
  • Shift + F5: הפעל מצגת מהשקופית הנוכחית
  • Ctrl + Shift + <: הקטן גודל גופן
  • Ctrl + Shift + >: הגדל גודל גופן

טיפים לשימוש יעיל בקיצורי דרך בעריכת מדיה ופרזנטציות

  • השתמש בקיצורי דרך לכלים הנפוצים ביותר כדי לחסוך זמן בעבודה.
  • למד את קיצורי הדרך לפעולות שאתה מבצע בתדירות גבוהה.
  • נסה ליצור הרגלים של שימוש בקיצורי דרך במקום להשתמש בעכבר.
  • התאם אישית קיצורי דרך בתוכנות שמאפשרות זאת לפי הצרכים שלך.
