חזרה לדף הראשי

קיצורי דרך למערכות הפעלה

Windows

  • Win + D: הצג/הסתר שולחן עבודה
  • Win + E: פתח סייר קבצים
  • Win + L: נעל את המחשב
  • Alt + Tab: החלף בין חלונות פתוחים
  • Win + Shift + S: צלם מסך חלקי
  • Ctrl + Shift + Esc: פתח את מנהל המשימות
  • Win + I: פתח הגדרות
  • Win + R: פתח את חלון "הפעלה"
  • Win + ימין/שמאל/למעלה/למטה: סדר חלונות
  • Win + Ctrl + D: צור שולחן עבודה וירטואלי חדש

macOS

  • Cmd + Space: פתח Spotlight
  • Cmd + Tab: החלף בין יישומים
  • Cmd + Q: סגור יישום
  • Cmd + W: סגור חלון
  • Cmd + Shift + 3: צלם מסך מלא
  • Cmd + Shift + 4: צלם מסך חלקי
  • Cmd + Shift + 5: פתח כלי צילום מסך
  • Cmd + Option + Esc: כפה סגירת יישום
  • Cmd + Control + F: מסך מלא
  • Cmd + H: הסתר יישום נוכחי

טיפים לשימוש יעיל בקיצורי דרך

  • התחל עם קיצורים בסיסיים ותרגל אותם עד שיהפכו לטבע שני.
  • נסה להשתמש בקיצורי מקלדת במקום בעכבר ככל האפשר.
  • חפש קיצורים ספציפיים לתוכנות שאתה משתמש בהן באופן קבוע.
  • שקול ליצור קיצורי דרך מותאמים אישית למשימות שאתה מבצע לעתים קרובות.
  • השתמש בתוכנות לתרגול קיצורי מקלדת כדי לשפר את המיומנות שלך.
