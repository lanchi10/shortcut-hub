חזרה לדף הראשי

קיצורי דרך לדפדפנים

Google Chrome

  • Ctrl + T: פתח כרטיסייה חדשה
  • Ctrl + W: סגור כרטיסייה נוכחית
  • Ctrl + Shift + T: פתח מחדש כרטיסייה שנסגרה
  • Ctrl + Tab: עבור לכרטיסייה הבאה
  • Ctrl + Shift + Tab: עבור לכרטיסייה הקודמת
  • Ctrl + L: סמן את שורת הכתובת
  • Ctrl + R: רענן את הדף הנוכחי
  • Ctrl + Shift + N: פתח חלון גלישה בסתר

טיפים לשימוש יעיל בקיצורי דרך בדפדפנים

  • השתמש ב-Ctrl + מספר כדי לעבור ישירות לכרטיסייה ספציפית (לדוגמה, Ctrl + 1 לכרטיסייה הראשונה).
  • לחץ על F6 כדי לעבור בין שורת הכתובת, סרגלי הכלים והתוכן של הדף.
  • השתמש ב-Ctrl + F כדי לחפש טקסט בדף הנוכחי.
  • לחץ על Ctrl + D כדי לשמור את הדף הנוכחי במועדפים.
  • השתמש ב-Ctrl + Shift + Delete כדי לפתוח את חלון ניקוי היסטוריה ומטמון.
