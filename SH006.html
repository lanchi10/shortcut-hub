קיצורי דרך לכלי AI

ChatGPT

כלי מבית OpenAI שמבוסס על מודלים של למידת מכונה לשיח אינטליגנטי.

  • Ctrl + /: פתח תפריט קיצורי דרך
  • Ctrl + Shift + C: העתק את כל השיחה
  • Ctrl + Shift + V: הדבק טקסט ושלח
  • ↑: ערוך את ההודעה האחרונה שלך
  • Tab, ואז Enter: השלם את ההצעה
  • Esc: התמקד בתיבת הקלט

Midjourney

כלי יצירה אמנותית שמאפשר ליצור תמונות חדשות באמצעות AI.

  • /imagine: צור תמונה חדשה
  • /blend: ערבב מספר תמונות
  • /describe: תאר תמונה קיימת
  • U1, U2, U3, U4: שדרג גרסה ספציפית
  • V1, V2, V3, V4: צור וריאציה של גרסה ספציפית

DALL-E

כלי AI מתקדם ליצירת תמונות חדשות על פי טקסט שנכתב.

  • Ctrl + Enter: צור תמונה
  • Ctrl + Z: בטל פעולה אחרונה
  • Ctrl + Shift + Z: חזור על פעולה שבוטלה
  • Ctrl + S: שמור תמונה
  • Ctrl + C: העתק תמונה

Whisper

Whisper הוא מודל AI להמרת דיבור לטקסט, מהיר וקל לשימוש.

  • Ctrl + Shift + S: התחל המרת דיבור לטקסט
  • Ctrl + Shift + P: עצור המרת דיבור לטקסט
  • Ctrl + Shift + C: העתק את הטקסט שהומר
  • Ctrl + Z: בטל המרה אחרונה

Copilot

GitHub Copilot הוא כלי AI המסייע בכתיבת קוד בזמן אמת עבור מפתחים.

  • Ctrl + Enter: בקש הצעה להשלמת קוד
  • Ctrl + /: דלג להצעה הבאה
  • Tab: קבל את ההצעה הנוכחית
  • Esc: התעלם מהצעה

טיפים לשימוש יעיל בקיצורי דרך בכלי AI

  • השתמש בקיצורי דרך כדי לחסוך זמן בפעולות חוזרות ונשנות.
  • נצל את היכולות של AI להשלמת משימות מורכבות במהירות.
  • השתמש בקיצורי דרך לניווט מהיר בין אפשרויות שונות בכלי AI.
  • למד את הפקודות הייחודיות לכל כלי AI כדי למקסם את היעילות שלך.
